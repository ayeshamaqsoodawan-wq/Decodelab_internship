# Project 1 – Responsive Frontend Interface
**DecodeLabs Industrial Training Kit | Full Stack Development Internship**

## 📌 Overview
This is Project 1 of the DecodeLabs Industrial Training Kit — the "interface phase" of the internship.
The goal was to build a responsive frontend interface for a simple web application using only
HTML, CSS, and basic JavaScript — no frameworks.

The deliverable is **Drafting Board**, a simple task-board web app styled around an architectural
"blueprint" theme, demonstrating responsive, mobile-first design.

## 🎯 Project Goal
Create a responsive frontend interface for a simple web application that:
- Uses HTML, CSS, and basic JavaScript
- Adapts to different screen sizes (mobile, tablet, desktop)
- Has a clean, user-friendly UI

## 🛠️ Built With
- **HTML5** – semantic structure (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
- **CSS3** – Grid (macro layout) + Flexbox (micro components), `clamp()` for fluid typography
- **Vanilla JavaScript** – task add/complete/remove, filtering, mobile nav toggle
- No frameworks or libraries

## 📱 Responsive Breakpoints
- **Mobile-first** base layout (single column)
- **Tablet:** 768px and up
- **Desktop:** 1024px and up

## ✨ Features
- Add, complete, and remove tasks
- Filter tasks by status (All / Open / Complete)
- Live count of open vs. completed items
- Collapsible mobile navigation menu
- Accessible focus states and ARIA attributes

## 📂 File Structure

 How to Run
1. Clone or download this repository
2. Open `index.html` in any web browser
3. Resize the browser window to see the responsive layout in action
